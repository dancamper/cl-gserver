Internal Generic Function handle-message-user

Package:

CL-GSERVER.ACTOR-CELL

Syntax: 

  handle-message-user ACTOR-CELL MESSAGE WITHREPLY-P → result

Arguments and Values:

  • ACTOR-CELL : a T
  • MESSAGE : a T
  • WITHREPLY-P : a T
  • result : a T.

Description:

The user defined message handler. Effectively this calls the `handle-call' or `handle-cast' functions.
