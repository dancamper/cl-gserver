Macro with-submit-handler

Package:

CL-GSERVER.MESSAGEB

Syntax: 

  with-submit-handler (MSGBOX MESSAGE WITHREPLY-P TIME-OUT) &REST BODY ⇒ form

Arguments and Values:

  • MSGBOX : a T
  • BODY : a T
  • result : a FORM.

Description:

Macro to let the caller specify a message handler function. Use this instead of `submit'.
