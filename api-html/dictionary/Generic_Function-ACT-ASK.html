Generic Function ask

Package:

CL-GSERVER.ACTOR

Syntax: 

  ask ACTOR MESSAGE &KEY TIME-OUT → result

Arguments and Values:

  • ACTOR : a T
  • MESSAGE : a T
  • TIME-OUT : a T
  • result : a T.

Description:

Sends a message to the `actor'. `ask' is synchronous and waits for a result. Specify `timeout' if a message is to be expected after a certain time. An `:handler-error' with `timeout' condition will be returned is the call timed out.
