Internal Function %actor-of

Package:

CL-GSERVER.ACTOR-SYSTEM

Syntax: 

  %actor-of SYSTEM CREATE-FUN DISPATCH-TYPE &KEY CONTEXT-KEY QUEUE-SIZE → result

Arguments and Values:

  • SYSTEM : a T
  • CREATE-FUN : a T
  • DISPATCH-TYPE : a T
  • CONTEXT-KEY : a T
  • QUEUE-SIZE : a T
  • result : a T.

Description:

Private API to create system actors. Context-key is either `:internal' or `:user' Users should use `actor-of'.
