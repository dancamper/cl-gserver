Function make-actor-system

Package:

CL-GSERVER.ACTOR-SYSTEM

Syntax: 

  make-actor-system &KEY &KEY SHARED-DISPATCHER-WORKERS → result

Arguments and Values:

  • &KEY : a T
  • SHARED-DISPATCHER-WORKERS : a T
  • result : a T.

Description:

Creates an `actor-system'. Allows to configure the amount of workers for the `shared-dispatcher'.
