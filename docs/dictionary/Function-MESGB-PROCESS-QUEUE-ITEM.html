Internal Function process-queue-item

Package:

CL-GSERVER.MESSAGEB

Syntax: 

  process-queue-item MSGBOX ITEM → result

Arguments and Values:

  • MSGBOX : a T
  • ITEM : a T
  • result : a T.

Description:

The `time-out' handling in here is to make sure that handling of the message is 'interrupted'. This should happen in conjunction with the outer time-out in `submit/reply'.
