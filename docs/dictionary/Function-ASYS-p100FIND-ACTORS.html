Internal Function %find-actors

Package:

CL-GSERVER.ACTOR-SYSTEM

Syntax: 

  %find-actors SYSTEM TEST-FUN &KEY &KEY CONTEXT-KEY → result

Arguments and Values:

  • SYSTEM : a T
  • TEST-FUN : a T
  • &KEY : a T
  • CONTEXT-KEY : a T
  • result : a T.

Description:

Private API to find actors in both contexts the actor-system supports. Users should use `find-actors'.
