Internal Generic Function handle-message

Package:

CL-GSERVER.ACTOR-CELL

Syntax: 

  handle-message ACTOR-CELL MESSAGE WITHREPLY-P → result

Arguments and Values:

  • ACTOR-CELL : a T
  • MESSAGE : a T
  • WITHREPLY-P : a T
  • result : a T.

Description:

The message handler which is usually called after the message was popped from a queue.

Known Documented Methods: 

  1.  handle-message  ACTOR-CELL (MESSAGE DELAYED-CANCELLABLE-MESSAGE) WITHREPLY-P → result

    We check here if the message is of type `delayed-cancellable-message', and if it got cancelled, in which case we respond just with `:cancelled'. 

  2.  handle-message  ACTOR-CELL MESSAGE WITHREPLY-P → result

    This function is submitted as `handler-fun' to message-box.
