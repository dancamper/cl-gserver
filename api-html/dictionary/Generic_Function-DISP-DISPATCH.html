Generic Function dispatch

Package:

CL-GSERVER.DISPATCHER

Syntax: 

  dispatch DISPATCHER DISPATCHER-EXEC-FUN → result

Arguments and Values:

  • DISPATCHER : a T
  • DISPATCHER-EXEC-FUN : a T
  • result : a T.

Description:

Dispatches a function (`dispatch-exec-fun') to a worker of the dispatcher to execute there. `dispatch' does a `ask' to a `dispatcher' worker, which means this call will block.
